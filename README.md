# ExtJS Test Task - Street Editor

Тестовое приложение на ExtJS с двумя связанными таблицами: города и улицы.

## Технологии
- ExtJS 6.0.0 (CDN)
- MVVM (ViewModel + ViewController)
- Хранилища в памяти (`Ext.data.Store` + `memory proxy`)

## Запуск
1. Клонировать репозиторий.
2. Открыть `index.html` в браузере.

## Реализовано
- Две параллельные таблицы (25%/75%).
- Таблица городов:
  - столбцы: город, регион, население;
  - фильтры: по городу (contains), по региону (combo, без произвольного ввода), по населению (>=);
  - динамические фильтры с триггером очистки;
  - пагинация.
- Таблица улиц:
  - фильтрация по выбранному городу;
  - столбцы: улица, компания, дома, примерное население, действие;
  - фильтры: по названию, по компании (combo c произвольным вводом и фильтрацией), по домам (точное совпадение), по населению (диапазон);
  - динамические фильтры с триггером очистки;
  - пагинация.
- Динамический расчет населения:
  - для улицы: `houses * 750`;
  - для города: сумма по его улицам;
  - в выводе используется префикс `~`.
- Действия:
  - удаление улицы через колонку действия с подтверждением;
  - редактирование строки (улица/компания/дома) с подтверждением сохранения;
  - добавление улицы через модальное окно с валидацией и кнопкой `Создать` только при валидной форме.

## Структура
- `index.html` - подключение ExtJS и классов приложения.
- `app.js` - bootstrap (`Ext.application` + `Viewport`).
- `app/data/SeedData.js` - исходные данные в памяти.
- `app/model/*.js` - модели (`City`, `Street`, `Company`).
- `app/store/*.js` - store-классы с memory proxy и paging.
- `app/view/main/Main.js` - основной экран с двумя таблицами.
- `app/view/main/MainController.js` - бизнес-логика и обработчики действий.
- `app/view/main/MainModel.js` - ViewModel и store-конфигурация.
- `app/view/street/AddStreetWindow.js` - окно добавления улицы.
